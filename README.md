# 트럼프 트윗과 주식시장 반응 분석  
## 언어적 감성과 주가 변동성의 관계

본 프로젝트는 도널드 트럼프 전 미국 대통령의 트위터 발언이  
미국 주식시장(S&P 500)에 단기적으로 어떤 영향을 미치는지를  
텍스트 마이닝(NLP)과 회귀 분석을 통해 실증적으로 분석한 연구 프로젝트입니다.

단순히 “트윗이 있었는가”가 아니라,  
**어떤 내용의 트윗이, 어떤 맥락에서, 언제 시장 반응을 유발하는지**에 초점을 둡니다.
---

## 1. 연구 배경 및 문제의식

트럼프 대통령의 트윗은 종종 금융시장을 움직인다는 인식을 낳아왔습니다.  
그러나 실제로 **트윗이 존재했다는 사실 자체만으로 시장이 반응하는지**에 대해서는  
정량적 검증이 충분하지 않습니다.

본 연구의 기초 분석 결과,
- 트윗 유무만으로는 시장 수익률의 유의미한 차이가 나타나지 않았습니다.

이에 따라 다음과 같은 문제의식에서 출발했습니다.

> “그렇다면 어떤 트윗이, 언제, 어떤 방식으로 시장을 움직이는가?”

---

## 2. 연구 질문 (Research Questions)

본 프로젝트는 다음 네 가지 질문에 답하고자 합니다.

1. 트럼프 트윗의 **존재 여부 자체**가 시장에 영향을 미치는가?
2. **특정 주제의 트윗**이 시장 반응을 유발하는가?
3. 트윗의 **감성(긍정/부정)** 이 시장 변동성 및 수익률에 영향을 미치는가?
4. **시기별(정책 국면별)** 로 트윗의 효과는 달라지는가?

---

## 3. 전체 분석 흐름

본 연구는 아래와 같은 단계적 분석 흐름을 따릅니다.

1. 트윗 전체 효과에 대한 기초 분석  
2. 트윗 내용 기반 주제 분석 (Topic Modeling – LDA)  
3. 무역정책 관련 주제 집중 분석  
4. 핵심 단어 수준의 시장 반응 검증  
5. 트윗 감성 분석  
6. 시기별 비교 및 종합 해석

## 4. 데이터 설명

- 장중 데이터(09:30–16:00 ET)만 사용

## 5. 분석 방법 및 Framework

### 5.1 트윗 전체 효과 분석
- 트윗 유무에 따른 평균 수익률 비교
- 트윗 이벤트 더미 변수(tweet_dummy) 사용
- 여러 시간 지평(1, 10, 20, 30, 45, 60, 120분)에서 분석

### 5.2 트윗 내용 기반 주제 분석
- 트윗 텍스트 전처리:
  - URL, 특수문자 제거
  - 토큰화 및 stemming
  - 불용어 제거
  - Bigram 생성
- LDA 토픽 모델링 적용
- coherence score를 기준으로 토픽 수(K=20) 선정
- 각 트윗의 주제별 확률 산출

### 5.3 무역정책 주제 집중 분석
- 무역·관세 관련 주제 식별
- 해당 주제 트윗의 시장 반응 비교
- 수익률 및 변동성 변화 분석

### 5.4 핵심 단어 수준 검증
- 정책을 직접적으로 시사하는 핵심 단어 선정
- 일반 정치적 표현과 비교
- 단어 수준 결과와 주제 분석 결과의 일관성 검증

### 5.5 트윗 감성 분석
- VADER 기반 감성 점수(긍정/중립/부정) 산출
- 감성별 시장 반응 비교
- 단기 수익률 및 변동성 영향 분석

### 5.6 시기별 비교 분석
- 정책 국면별 데이터 분리
- 2017년 집중 분석
- 트윗 효과의 시기적 변화 해석

---

## 6. 분석 결과 요약

- 트윗 전체 효과는 대부분의 시간 구간에서 통계적으로 유의하지 않음
- 특정 주제(무역·관세 관련) 트윗에서 단기 시장 반응이 관측됨
- 부정적 감성의 트윗은 변동성 확대와 연관
- 트윗 효과는 시기에 따라 다르게 나타남
